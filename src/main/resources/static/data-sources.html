Data Sources

The FNA platform for G20 runs on a secure cloud, and encompasses a set of technologies for sourcing unstructured and structured data from a variety of sources (e.g data vendors and social media) and relates this data to relevant supervisory datasets.

For this Global Financial System Monitor, our platform is utilising data from the below sources. In the future these can be combined with supervisory data sets for additional insights.
